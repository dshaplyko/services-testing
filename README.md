# web-services-testing

Test framework for web-services testing.

## Requirements

* Git
* NodeJS 10 or higher

## Setup

```
$> git clone https://github.com/dshaplyko/services-testing.git
$> cd web-services-testing
$> npm install
```
## Running tests

To run tests you should execute the command 

```
$> npm run test
```
